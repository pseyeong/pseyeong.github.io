tailwind

  • tailwind설치

    1. 프로젝트 폴더 생성

    2. [ Bash - Terminal ] npm init -y

    3. [ Bash - Terminal ] npm install -D tailwindcss @tailwindcss/cli

    4. css 파일 생성 후 @import "tailwindcss"; 추가

    5. [ Bash - Terminal ] npx @tailwindcss/cli -i ./src/input.css -o ./dist/output.css --watch

    ** 폴더 경로 맞추기 **

    6. 공통으로 사용되는 html 파일에 추가

  • 터미널에서 npm run watch로 실시간 적용 방법

    package.json에 scripts 추가

    [ Bash - Terminal ]
    개발용 자동 감시 : npm run watch
    배포용 빌드 : npm run build
