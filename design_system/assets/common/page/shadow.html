Shadows

그림자는 총 3개의 스타일로 이루어져 있습니다. 각 섀도우는 특정 배경색에 최적화되어 있으므로 배경색에 맞춰 섀도우를 선택하는 것이 좋습니다.
그림자의 위치는 항상 Top view를 유지합니다.

shadow_s 흰색 배경의 작은 물체에 사용되는 가장 작은 그림자입니다. e.g. Small button, Chips, Badge

shadow_m 흰색 배경의 중간 크기 개체에 사용되는 중간 크기 그림자입니다. e.g. Mid/Large button

shadow_l 화면 너비의 50%를 초과하는 흰색 배경의 큰 개체나 강조가 필요한 구성 요소에 사용되는 큰 크기의 그림자입니다. e.g. Large card, Bottom Sheet
