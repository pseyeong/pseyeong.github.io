Box Button 박스 형태의 가장 일반적인 버튼입니다. 콘텐츠 영역이나 화면 하단 영역에 배치되어 화면에서 중요한 동작을 실행하는 버튼으로 사용됩니다.

* 활성화 : active / 비활성화 : disabled 클래스 추가

CTA Button

사용자 상호 작용을 유도하기 위한 클릭 가능한 요소로, 화면 내 링크 및 기능, 사용자 액션이 필요한 부분에 사용합니다.

Primary

가장 중요하거나 사용자의 눈에 띄어야 하는 경우에 사용하며, 한 페이지에 한 번만 사용하는 것을 권장합니다.

XL

Default Hover Pressed Disabled

L

Default Hover Pressed Disabled

M

Default Hover Pressed Disabled

S

Default Hover Pressed Disabled

Secondary

중요하거나 사용자의 눈에 띄어야 하지만, 페이지 내에 기본 버튼이 이미 존재하는 경우에 사용합니다.

XL

Default Hover Pressed Disabled

L

Default Hover Pressed Disabled

M

Default Hover Pressed Disabled

S

Default Hover Pressed Disabled

Tertiary

눈에 덜 띄어야 하는 경우에 사용하며, 독립적인 행동을 위한 경우에도 사용됩니다.

XL

Default Hover Pressed Disabled

L

Default Hover Pressed Disabled

M

Default Hover Pressed Disabled

S

Default Hover Pressed Disabled

Outlined

화면 내 기능을 컨트롤 할 때 사용합니다. 기본 버튼이 있는 페이지의 보조 작업에도 사용할 수 있습니다. (취소, 닫기 등)
Outlined 버튼은 Solid (Primary / Secondary / Tertiary) 버튼보다 우선순위가 낮은 작업 버튼으로 사용됩니다.

XL

Default Hover Pressed Disabled

L

Default Hover Pressed Disabled

M

Default Hover Pressed Disabled

S

Default Hover Pressed Disabled
