button - Overview 설명

Icon Type

  • CTA Button

    Box Button

    화면 내 링크 및 기능, 사용자 액션이 필요한 부분에 사용합니다.

  • Outlined

    Box Button

    화면 내 기능을 컨트롤 할 때 사용합니다. 기본 버튼이 있는 페이지의 보조 작업에도 사용할 수 있습니다. (취소, 닫기 등)

  • Ghost

    Text Button

    버튼의 작업이 화면 전체 내용과 큰 관련이 없거나 관심도가 낮은 경우에 사용됩니다.

  • Link

    Text Button

    페이지 내 중요도가 낮은 외부 링크에 사용됩니다.

  • Icon Button

    Icon Button

    공간이 충분하지 않거나 아이콘이 사용자가 이해할 수 있을 만큼 직관적인 메타포인 경우, 텍스트 없이 아이콘 버튼을 사용할 수 있습니다.

  • FAB

    FAB

    플로팅 액션 버튼은 스크롤에 영향 없이 화면 내 고정된 위치에 노출되는 컴포넌트 입니다.

  • Pill FAB

    FAB

    버튼 위치의 고정이 필요하지만, FAB버튼을 사용할 수 없는 경우에 사용되며, 새로고침, 펼쳐보기와 같은 추가 작업에도 사용할 수 있습니다.

  • Segmented
    Button

    Segmented Buttons

    내용을 구분할 때 사용하는 버튼형 컴포넌트 입니다. 선택한 버튼에 따라 하단 내용이 달라집니다.
